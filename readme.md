- get
- put : api body =
  ```javascript
  {
  "id": 2,

  "desc" : "updated description 2"
  }
  ```

  ```javascript
  {
  "id": 1,
  "title": "Note 1 ",
  "desc": "Description 1"
  }
  ```
- delete: put notes id in the api params
